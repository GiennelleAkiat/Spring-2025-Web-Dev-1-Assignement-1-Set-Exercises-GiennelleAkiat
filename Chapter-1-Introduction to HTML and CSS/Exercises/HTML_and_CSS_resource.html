Horizontal line

To create a horizontal line, you use <hr> on its own, it has no closing tag

Text formating

You can format text to be bold, italic, & underlined using these tags: <b>, <i>, <u>

Link

The <a href> tag is for making a clickable link. For example: <a href="url">Link</a>

Images

Use the <img src=""> tag to add images. Within the quotations, you add the image source using either a link or the image name and format if that image is within the same file of the script.
For example: <img src = "https://pbs.twimg.com/profile_images/1743456442378838016/1F8pRFPX_200x200.jpg", alt="Flug_The_Alien">

Lists

Creating lists use the <ul> for unorder lists and <ol> for ordered lists
Adding items within the lists uses the <li> tag

Div

The <div> tag can't be seen on the webpage but they are used to organize sections of the webpage, making it easier to format.

CSS

CSS full form means Cascading Style Sheet, which makes styling and formating webpages easier to do. It can be done within the .html file or as a seperate .css file.
